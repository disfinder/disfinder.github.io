# Finance

## Cashback

Chasing cashback, miles, points — is a naive and mostly useless pursuit. Of course, there is some financial benefit to it. Whether it is always worth the effort — not at all. From this perspective, a separate card with 5% cashback on Amazon makes sense if you set it up and forget about it. Tracking and hunting for cashback is almost always wasted time.

## Discipline

The second-to-last weekend of the month is the financial one.
Make sure all payments are done, import bank statements into accounting.
