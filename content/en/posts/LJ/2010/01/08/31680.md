Updated the firmware to the latest version.
Before updating, I noticed with some surprise that this router had disappeared from the official website — in its place there is now some DIR-300/NRU, which looks different. Fortunately, the firmware files were preserved on ftp.dlink.ru, so I gave it a try.
The goal was to improve the Wi-Fi transfer speed, which on version 1.04 wouldn't even reach 1 MB/s. After the update things got a bit better — the speed to a nearby (city-local) server increased and reached 2–3 MB/s. However, the transfer speed between two laptops, each connected via Wi-Fi to this now-updated router, improved only marginally and still doesn't exceed 1.3 MB/s. I'll try updating the drivers too, but I still suspect the router itself — I think it just can't keep up.
Which, in principle, is not surprising or alarming given what it cost. (By the way, this is the only purchase I've made that actually got more expensive after I bought it. Usually everything I buy drops significantly in price within a month or two. But here — what a surprise :)
